# AI Usage and Reflection

## 1. How did you use AI while completing this assignment?

I used AI to help plan the basic SwiftUI structure, connect the Cleveland Museum of Art Open Access API, and identify the minimum model fields needed for searching and displaying artworks. AI also helped generate first versions of the views, service layer, image handling, persistent favorites, and simple error/loading states.

## 2. What did you learn from using AI on this project?

I learned how the ideas from the TheMet tutorial can transfer to another public museum API. The most useful part was seeing how JSON fields from the Cleveland Museum API can be mapped into Swift models and then displayed in a NavigationStack with SwiftUI views.

## 3. What would you improve or verify next?

Next, I would keep improving the visual design, add more detailed filters, test more search terms, and review the API documentation again to confirm which fields are best for the final version of the app. I would also verify the final GitHub repository link before submitting it to the instructor.
